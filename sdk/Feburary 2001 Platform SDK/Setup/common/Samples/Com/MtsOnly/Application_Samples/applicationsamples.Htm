Application Samples

Overview

In addition to smaller samples demonstrating individual features, we also provide "best practices" application samples that show you how to use different services in one application. Island Hopper is a "best practices" application that allows users to enter ads in an on-line classifieds section and newspaper employees to review and bill for the advertisements.
