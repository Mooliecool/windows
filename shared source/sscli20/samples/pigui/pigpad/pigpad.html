PigPad Sample

Sample Overview

This sample implements a simple Notepad editor using the Tk wrapper assembly, sharedsourcecli.tk.dll.

This sample requires that Tcl/Tk be installed on your computer in order to function.

Sample Source and Build Output Locations

The sample source is found in sscli20\samples\pigui\pigpad. 

The source file is:

  • pigpad.cs

The build output location is %_NTTREE%\samples\pigui\pigpad.  The output file is an executable assembly named pigpad.exe.

Building the Sample

All samples are built from the buildall script. 

You can also build all the samples by switching to the root of the sample directory, sscli20\samples, and typing build -c.

You can build this specific sample by switching to the sample directory and typing build -c.

Running the Sample

These steps require that the Shared Source CLI (SSCLI) be already built and functional.

  1. Run env.bat.
  2. Switch to the %_NTTREE%\samples\pigui\pigpad directory.
  3. Type the following command:

    clix pigpad.exe
