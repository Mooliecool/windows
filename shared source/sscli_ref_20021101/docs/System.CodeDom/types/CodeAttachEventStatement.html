System.CodeDom.CodeAttachEventStatement Class

Assembly: System.dll
Namespace: System.CodeDom
Summary
Represents a statement that attaches an event handler.
C# Syntax:
[Serializable]
public class CodeAttachEventStatement : CodeStatement
Remarks
CodeAttachEventStatement can be used to represent a statement that adds a listener or handler for an event or delegate.
Example
The following code creates a CodeAttachEventStatement named attachevent1 that attaches an event handler method named UpdateDisplay to an event named OnPaint: 
          CodeAttachEventStatement attachevent1 = new CodeAttachEvent(this, OnPaint, UpdateDisplay);
