System.Reflection.Emit.PropertyToken Structure

Assembly: Mscorlib.dll
Namespace: System.Reflection.Emit
Summary
The PropertyToken class is an opaque representation of the Token returned by the metadata to represent a property.
C# Syntax:
[Serializable]
public struct PropertyToken
Thread Safety
Reflection Emit is thread-safe when using assemblies that were created with the AppDomain.DefineDynamicAssembly method with the Boolean parameter isSynchronized set to true.
